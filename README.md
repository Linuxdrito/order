# order

`order` es un administrador de archivos para la terminal escrito en C++.

## Características

- Interfaz de tres paneles.
- Navegación mediante teclado.
- Previsualización de archivos de texto.
- Búsqueda recursiva.
- Copiar, mover, renombrar y eliminar archivos.
- Apertura de archivos con programas externos.

## Dependencias

- Compilador compatible con C++17.
- Sistema POSIX.


## Controles

| Tecla | Acción |
|-------|--------|
| ↑ ↓ | Mover selección |
| ← → | Volver / Entrar a un directorio |
| Enter | Abrir archivo |
| / | Buscar |
| : | Modo comando |
| c | Copiar |
| m | Mover |
| p | Pegar |
| h | Ayuda |
| q | Salir |
